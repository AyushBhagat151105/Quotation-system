<h1>Quotation System</h1>

This Repo made for to automate quotation workflow between admin(user) and client.

```cmd
pnpm dev:frontend
```

use this script to start frontend.

```cmd
pnpm dev:backend
```

use this script to start backend

<h3>ENV for backend</h3>

```.env
PORT
LOCAL_IP
DATABASE_URL
JWT_SECRET
MAILTRAP_HOST
MAILTRAP_PORT
MAILTRAP_USERNAME
MAILTRAP_PASSWORD
MAILTRAP_FROM
APP_URL
```

<h3>ENV for frontend</h3>

```.env
VITE_API_URL
```
